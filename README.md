 Customer Churn Analysis (Power BI)

🎯 Objective

Analyze customer churn patterns and identify key factors driving customer attrition.

 🛠 Tools Used

* Power BI
* Excel

 📊 Dashboard Features

* Churn Rate & Retention KPIs
* Churn by Subscription Type
* Churn by Tenure
* Usage vs Churn Analysis
* Region-wise Churn (Map)

 🔍 Key Insights

* New customers have the highest churn rate
* Basic plan users churn more than premium
* Low usage customers are more likely to churn

 💡 Recommendations

* Improve onboarding for new users
* Provide incentives for high-risk customers
* Increase engagement through targeted campaigns
